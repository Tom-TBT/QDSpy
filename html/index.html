Welcome to QDSpy’s documentation!¶

This is a simple Python software for scripting and presenting stimuli for visual neuroscience. It is based on QDS, uses OpenGL and primarly targets Windows but may also run on other operating systems.

Note that this is an early, experimental (alpha) version. See also Disclaimer.
